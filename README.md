# Visualização de Dados — UFF 2026.1

Repositório contendo os trabalhos desenvolvidos para a disciplina **Visualização de Dados** do Instituto de Computação da Universidade Federal Fluminense (UFF), ministrada pelo Prof. Marcos Lage.

## Tecnologias principais

* D3.js
* DuckDB

---

## Trabalho 1 — Redesign do TaxiVis

Redesign de visualizações do dataset de corridas de táxi de Nova Iorque, inspirado no sistema TaxiVis.

## Trabalho 2 — Sistema Interativo de Criminalidade do Estado do Rio de Janeiro

Sistema de visualização interativa para exploração de estatísticas criminais do estado do Rio de Janeiro, utilizando mapas coropléticos e visões coordenadas.

## Execução local

Instale as dependências:

```bash
npm install
```

Inicie o servidor local:

```bash
npm run dev
```

A aplicação estará disponível em:

```text
http://localhost:5173
```
